DNAstack

Title: Java Backend Developer Intern
Date: May 2018 - August 2018

Summary

Through the Google Summer of Code (GSoC) program, I was accepted to spend the summer working on an open source project for the organization Global Alliance for Genomics and Health (GA4GH). The project was titled "Global Filesystem and Search Engine for Genomics Data", whose goal was to create a generic API on top of existing object storage systems so workflow systems can access data in a single, standard way regardless of where it's stored.

GA4GH have developed the Data Object Service (DOS), which is an emerging API standard for specifying location of data across different cloud environments. I created is an implementation of a DOS Server, which hosts and allows the discovery of data objects. By completing this project I learned how to use the Springboot framework to build an API, how to connect a MYSQL database to an API, how to connect Keycloak authentication to an API, how to package it using Maven, and how to integrate a backend such as the DOS Server to a UI.
